# Git Switch

Switch git users quickly.

## Installation

#### MacOS

```bash
$ brew install git-switch
```

#### Ubuntu

```bash
$ apt-get install git-switch
```

## Usage

#### List users

```bash
$ git-switch list
```

#### Add user

```bash
$ git-switch add
```

#### Switch user
Switch user for current repository

```bash
$ git-switch add
```

#### Reset
Remove all users

```bash
$ git-switch reset
```
